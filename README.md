# 📅 March Projects – Embedded Systems & IoT

This repository contains a collection of Arduino / Embedded Systems mini-projects completed in March. Each project focuses on real-world applications using sensors, actuators, and microcontrollers.

---

## 📌 Project 11: Smart Door Automation

### 🧠 Objective
Design an automated door system that detects a user and opens/closes the door.

### ⚙️ Components
- IR Proximity Sensor  
- Servo Motor  
- Microcontroller (Arduino)  
- Jumper Wires  

### 🔄 Working
- IR sensor detects human presence.
- Signal sent to Arduino.
- Servo motor rotates to open the door.
- After a delay, the door closes automatically.
- Events can also be logged (optional).

### 💡 Applications
- Smart homes  
- Offices  
- Security systems

## 📌 Project 12: Forklift Safety Alert

### 🧠 Objective
Enhance safety by detecting nearby pedestrians and stopping the forklift.

### ⚙️ Components
- Ultrasonic Sensor  
- Buzzer  
- Motor (or simulation)  
- Arduino  

### 🔄 Working
- Ultrasonic sensor measures distance.
- If an object is within 2 meters:
  - Buzzer activates  
  - Motor stops  
- Otherwise, the system runs normally.

### 💡 Applications
- Industrial safety  
- Warehouses  
- Automation systems

## 📌 Project 13: Vending Machine Dispenser

### 🧠 Objective
Create a touchless vending system using a proximity sensor.

### ⚙️ Components
- Capacitive Proximity Sensor  
- Relay Module  
- Arduino  
- EEPROM (for storage)  

### 🔄 Working
- Sensor detects hand presence.
- Relay triggers the dispensing mechanism.
- Inventory count decreases and is stored in EEPROM.

### 💡 Applications
- Smart vending machines  
- Contactless systems  
- Retail automation

## 📌 Project 14: Assembly Line Object Counter

### 🧠 Objective
Count metal objects on a conveyor belt automatically.

### ⚙️ Components
- Inductive Sensor  
- LCD Display  
- Arduino  

### 🔄 Working
- Inductive sensor detects metal objects.
- Each detection increments the counter.
- Data is displayed on the LCD in real-time.

### 💡 Applications
- Manufacturing industries  
- Quality control systems  
- Production monitoring

## 📌 Project 15: Parking Slot Occupancy System

### 🧠 Objective
Monitor parking slot availability and transmit data wirelessly.

### ⚙️ Components
- PIR Sensor  
- Bluetooth Module  
- Arduino  

### 🔄 Working
- PIR sensor detects vehicle movement.
- Updates parking slot status (occupied/free).
- Sends data via Bluetooth to a mobile/device.

### 💡 Applications
- Smart parking systems  
- IoT-based traffic management  
- Urban planning
